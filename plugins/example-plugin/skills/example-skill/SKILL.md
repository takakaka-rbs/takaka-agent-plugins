# example-skill

サンプルスキルです。このスキルはプラグイン構成の参考例として機能します。

## Usage

```
/example-skill [message]
```

## Instructions

1. 引数 `$ARGUMENTS` を受け取る
2. 引数の内容をそのままエコーして返す

## Examples

```
/example-skill hello world
→ You said: hello world
```
